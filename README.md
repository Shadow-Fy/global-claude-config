# global-claude-config

[Claude Code](https://claude.ai/code) 全局配置。

## 内容

- `CLAUDE.md` — 全局行为指令
- `commands/` — 自定义命令
- `skills/` — 自定义技能
- `plugins.json` — 推荐插件清单

## 使用

```bash
# 克隆配置
git clone https://github.com/Shadow-Fy/global-claude-config.git %USERPROFILE%\.claude

# 安装插件（根据 plugins.json）
claude plugin install <plugin-name>
```
